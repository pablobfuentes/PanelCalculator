# Panel Calculator

Solar panel layout, tributary load, FEA, and BOM calculator (Streamlit + PyNite).

## Setup

```bash
python -m venv venv
venv\Scripts\activate          # Windows
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
