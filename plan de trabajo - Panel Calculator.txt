Phase 2 - 
Data model
Define a PanelSpec dataclass: length, width, weight, tilt_angle

Define a LayoutConfig dataclass: mid_clamp_gap, alley_width, max_area_x, max_area_y
Layout logic

Build pair_panels() — returns a bounding box for 2 panels including the 1" mid-clamp gap

Build accumulate_row() — spaces pairs with alley every 2 pairs (4 panels), returns list of (x, y, w, h) rectangles

Build accumulate_grid() — repeats rows in Y direction with alley insertion

Build fit_to_area() — reduces panel count step-by-step until grid fits within max_area_x × max_area_y
Validation

Write unit tests for each function above using pytestlibrary

Generate a Plotly figure showing panel rectangles, alley gaps, and bounding box — confirm visually with a known layoutkey
